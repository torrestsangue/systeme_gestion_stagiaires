# Devcode WhatsApp API - Module 1 (Authentification)

Ce projet est la plateforme de gestion d'envoi de messages WhatsApp de **Devcode Cameroun**. 
Ce module gère l'inscription, la connexion (JWT) et la gestion des comptes entreprises.

## 🛠 Prérequis

Avant de commencer, assurez-vous d'avoir installé :
- **Node.js** (Version 18 ou plus)
- **MariaDB** ou **MySQL**
- **npm** (installé avec Node.js)
